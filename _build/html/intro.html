Experimentation I 2022-2023¶

Note

Entry requirements Only open to MSc Psychology (research) students

Basic overview of course¶

First 2 weeks:¶

basic introduction into Python, learning to use variables, functions, decision-making statements, loops. Remainder: during each practical session, students implement one or more experimental paradigms in OpenSesame using Python code Examples of paradigms (research from all units represented, similar to those covered in E-Prime introduction book The E-Primer):

Simple reaction time¶

Simple RT in OpenSesame Python: RT analyse, split per block, outlier removal, plotting figure, histogram in python Python: Red/Green proc: If then in python

Python: Feedback display (if-then, ) Random jitter
