Python exercises: Dataframes¶

Exercise 1¶

In the tutorial you followed along an explanation using a open dataset with information on salary. There are many more publicly available datasets you can directly use. For this exercise, you can choose a dataset you find interesting. The website Our World in Data has datasets on the most pressing world-problems, now you are learning the tools how to work with these datasets! Pretty cool right. In your own Python IDE, choose and download a dataset you find interesting (search for the .csv file) and replicate each step from the the tutorial, starting from subchapter 6.6. Below is the code to get you on your way.

A dataset we recommend exploring is the dataset on global CO2 and greenhouse emission. However, you are free to choose any dataset you find interesting.
